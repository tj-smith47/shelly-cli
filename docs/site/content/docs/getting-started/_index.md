Welcome to Shelly CLI! This guide will help you install the CLI and start controlling your Shelly smart home devices in minutes.

## Quick Navigation

- [Installation](/docs/getting-started/installation/) - Install on macOS, Linux, Windows, or Docker
- [Quick Start](/docs/getting-started/quickstart/) - Get up and running in 5 minutes
- [First Device](/docs/getting-started/first-device/) - Detailed guide to adding your first device

## Requirements

- A Shelly device on your local network
- Network access to the device (same subnet or routed)
- For Gen2+ devices: Device IP address or mDNS name
- For Gen1 devices: Device IP address

## Supported Devices

Shelly CLI supports **all** Shelly device generations:

| Generation | Examples | Notes |
|------------|----------|-------|
| **Gen1** | Shelly 1, Shelly 2.5, Shelly Plug | ESP8266-based, HTTP API |
| **Gen2** | Shelly Plus 1, Plus 2PM, Pro 4PM | ESP32-based, RPC API |
| **Gen3** | Shelly Plus 1 Mini, i4 DC | Latest ESP32, RPC API |
| **Gen4** | Shelly Wall Display | Advanced features, RPC API |
