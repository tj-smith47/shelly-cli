shelly debug

Debug and diagnostic commands

Synopsis

Debug and diagnostic commands for troubleshooting Shelly devices.

These commands provide low-level access to device communication protocols and diagnostic information. Use them for debugging issues or exploring device capabilities.

WARNING: Some debug commands may affect device behavior. Use with caution.
