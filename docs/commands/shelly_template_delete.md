## shelly template delete

Delete a template

### Synopsis

Delete a saved template permanently.

```
shelly template delete <template> [flags]
```

### Examples

```
  # Delete a template (with confirmation)
  shelly template delete my-template

  # Delete without confirmation
  shelly template delete my-template --yes

  # Using alias
  shelly template rm my-template
```

### Options

```
  -h, --help   help for delete
  -y, --yes    Skip confirmation prompt
```
